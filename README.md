### About

I'm **Shresth Samyak**, an **AI Engineer** working on **computer vision**, **LLM inference optimization**, and **security for AI agents (Model Context Protocol)**. Most of my work sits at the boundary between research and production — taking a model or protocol that works in a notebook and making it safe, fast, and cheap enough to ship. I work day to day in **Python** with **PyTorch** and **JAX**. Based in India, open to AI engineering roles and collaborations.

**Focus areas:** LLM inference and token optimization · Retrieval-Augmented Generation (RAG) · Model Context Protocol (MCP) and agent security · Computer Vision · applied ML on constrained hardware.
